About This Project

Storytel Player for Desktop is an unofficial, open-source desktop application that provides a native experience for enjoying Storytel audiobooks. This project is not affiliated with or endorsed by Storytel.

The application features a compact UI optimized for desktop use, with system tray integration and support for all major platforms including Windows, macOS (Intel & Apple Silicon), and Linux (x64, ARM64, ARMv7).

⚠️ Important: This is an independent project and is not affiliated with Storytel. A valid Storytel subscription is required to use this application.
